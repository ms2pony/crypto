## avx2大整数模乘的limb可以超过26bit

无论在什么时候avx2形式的大整数的limb都可以为28-bit长，所以要得到256-bit长的结果还需要进行模p运算。

## c 编程-note

### 函数形参的问题

```c
int a(int *i){
  i->a++;
  i->a++;
  // 修改形参`i`的值，实参`i`不发生改变
  // 但修改`i`的成员如`i->a++`时，实参`i`的`i->a`发生改变
}
```

应用，要通过函数修改一个变量`a`，函数的形参应该为地址形式如`&a`，传入的实参为`&a`即 a 的地址；修改数组 a 的元素时，
直接传向函数传入数组名`a`即可
